Digital Humanities researcher, librarian, and archivist specializing in classical Arabic philology, TEI-XML documentary encoding, and Semantic Web knowledge representation. My work bridges material manuscript culture and computational ontology engineering to model historical texts, esoteric lexicons, and Linked Open Data networks.

## 📌 Featured Project

### **Mapping the Boundary: A Bilingual Digital Documentary Edition and Semantic Knowledge Graph of the Kitāb al-Ḥudūd (Kr 780)**

A Digital Humanities dissertation project establishing a machine-readable, open-access digital framework for the foundational lexicographical treatise of the *Corpus Jabirianum*.

* **Philological Baseline:** Producing the first scholarly English translation and a TEI-XML Documentary Edition based on the primary witness (*MS Cairo, ṭabīʿiyyāt 114*), auditing Paul Kraus’s 1935 printed edition to restore scribal variance and paratextual data.
* **Lexicographical Triangulation:** Decoding Jābir's native ontology by combining classical root extraction (*Lane*), post-classical scientific/esoteric terminology (*Dozy*), and legacy French interpretations (*Kazimirski*).
* **Spatial IIIF Mapping:** Linking parallel Arabic-English TEI segmentation directly to digital manuscript surrogates using W3C Web Annotation Data Model (WADM) bounding box coordinates.
* **Knowledge Graph Engineering:** Formalizing the technical lexicon into a queryable Semantic Web ontology mapped to CIDOC-CRM, SKOS, and OWL, verified locally via Apache Jena Fuseki and published through Omeka S and Zenodo.

## 🛠️ DH Pipeline & Technical Stack

* **Textual Encoding & Philology:** TEI-XML, oXygen XML, Custom Schemas (ODD/RNG), Python (XML parsing & data extraction)
* **Spatial & IIIF Annotations:** IIIF Manifests, W3C Web Annotation Data Model (WADM), Mirador, Recogito
* **Semantic Web & Ontologies:** CIDOC-CRM, SKOS, OWL, Protégé, JSON-LD
* **Database & Querying:** Apache Jena (Fuseki), SPARQL
* **Infrastructure & Preservation:** Git/GitHub CI/CD, Omeka S, Zenodo (DOI Archiving)
